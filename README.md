# ClaudePersona

클로드 커스텀을 해보아요

## 소개

클로드에 다양한 캐릭터 페르소나를 씌워 대화하는 프로젝트입니다.
슬래시 커맨드 하나로 캐릭터를 전환할 수 있습니다.

## 설치 (전역 사용)

어느 디렉토리에서든 커맨드를 사용하려면 설치가 필요합니다.

1. 레포를 클론합니다.
2. 클론한 디렉토리에서 Claude Code를 열고 아래 커맨드를 실행합니다.

```
/install
```

이후 어떤 프로젝트에서도 슬래시 커맨드를 사용할 수 있습니다.

> 이미 설치된 파일은 덮어쓰지 않습니다. 안전하게 재실행할 수 있습니다.

## 커맨드 목록

| 커맨드 | 설명 |
|--------|------|
| `/char-<캐릭터명>` | 현재 대화에만 캐릭터 적용 (대화 종료 시 해제) |
| `/char-<캐릭터명>-on` | 캐릭터를 `~/.claude/CLAUDE.md`에 영구 등록 |
| `/char-off` | 영구 등록된 캐릭터 해제 |

## 프로젝트 구조

```
ClaudePersona/
├── .claude/
│   └── commands/
│       ├── install.md               ← 전역 설치 커맨드
│       ├── char-<캐릭터명>.md       ← 일회성 캐릭터 적용
│       ├── char-<캐릭터명>-on.md    ← 캐릭터 영구 등록
│       ├── char-off.md              ← 캐릭터 해제
│       └── private/                 ← 공개하지 않는 개인 캐릭터
├── personas/
│   └── <캐릭터명>.md               ← 캐릭터 원본 정의 파일 (참고용)
└── TEMPLATE.md                     ← 캐릭터 작성 템플릿
```

## 캐릭터 추가 방법

1. `TEMPLATE.md`를 참고해서 캐릭터를 정의합니다.
2. `personas/<캐릭터명>.md`에 정의 파일을 저장합니다.
3. `.claude/commands/char-<캐릭터명>.md`에 일회성 커맨드 파일을 만듭니다.
4. 영구 적용 기능이 필요하면 `.claude/commands/char-<캐릭터명>-on.md`도 만듭니다.
5. `/install`을 재실행하면 새 커맨드가 전역에 등록됩니다.
