아래 절차를 수행해줘.

1. `~/.claude/commands/` 디렉토리가 존재하는지 확인한다.
   - 존재하지 않으면 디렉토리를 새로 생성한다.

2. 현재 프로젝트의 `.claude/commands/` 안에 있는 `.md` 파일 목록을 확인한다.
   - `private/` 폴더 안의 파일은 제외한다.
   - `install.md` 파일 자신도 제외한다.

3. 각 파일에 대해:
   - `~/.claude/commands/`에 동일한 이름의 파일이 이미 존재하면 건너뛴다.
   - 존재하지 않으면 해당 파일을 `~/.claude/commands/`로 복사한다.

4. 작업 완료 후 아래 형식으로 결과를 보고한다.
   - 복사된 파일 목록
   - 이미 존재하여 건너뛴 파일 목록
   - "이제 어느 디렉토리에서든 슬래시 커맨드를 사용할 수 있습니다."
